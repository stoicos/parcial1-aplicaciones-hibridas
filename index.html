🎞Videoclub

API de películas y directores de cine

Creada en Node, Express y MongoDB

Requiere una creación y autenticación de usuario para acceder a más funcionalidades

Dentro de las funcionalidades que puede acceder el usuario está la creación, modificación y eliminación de documentos

Empezar

  • $ npm i
  • $ npm start

Lista de endpoints:

# Películas

El id del director es necesario para crear peliculas en el caso de POST

  • GET 127.0.0.1:3000/movies/
  • POST 127.0.0.1:3000/movies/:id
  • PUT 127.0.0.1:3000/movies/:id
  • DELETE 127.0.0.1:3000/movies/:id

# Directores

  • GET 127.0.0.1:3000/directors/
  • POST 127.0.0.1:3000/directors/
  • PUT 127.0.0.1:3000/directors/:id
  • DELETE 127.0.0.1:3000/directors/:id

# Usuarios

  • GET 127.0.0.1:3000/users/
  • POST 127.0.0.1:3000/users/
  • PUT 127.0.0.1:3000/users/:id
  • DELETE 127.0.0.1:3000/users/:id
